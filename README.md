# RESTinventory
Simple rest API built in Django Framework that execute C.R.U.D methods in a SQL database.

# Application pre-requisites
Make sure you have Python3 installed.  
You'll need **[Python >= 3.6](https://www.python.org/downloads/)** and **[django](https://www.djangoproject.com/download/)**  

# Instruction to run the app

***step 1:*** Clone the repository
```
git clone https://github.com/w4d3r0/RESTinventory.git
```

***step 2:*** Navigate to the project folder
```
cd RESTinventory/restinventory
```

***step 3:*** Run the migration
```
python3 manage.py migrate
```

***step 4:*** Run the app
```
python3 manage.py runserver
```

***step 5:*** Navigate to http://localhost:8000 on your browser
